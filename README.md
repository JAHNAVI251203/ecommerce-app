# ClickCart 🛒

A full-stack MERN e-commerce application built to understand how real online stores work end-to-end — product browsing, cart, checkout, payments, and order tracking.

**Live Demo:** https://clickcart-store.vercel.app

---

## Tech Stack

Frontend:
React, Vite, React Router

Backend:
Node.js, Express

Database:
MongoDB Atlas

Payments:
Razorpay (Test Mode)

Deployment:
Vercel

---

## Features

* Product browsing and search
* Cart management
* Razorpay checkout integration
* Order history
* Order tracking page
* Admin dashboard (users, orders, revenue)

---

## Architecture

React (Vite)<br>
↓<br>
Express API<br>
↓<br>
MongoDB Atlas<br>
↓<br>
Razorpay
